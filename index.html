  • This started as my final project for the Code Louisville Front-End Web Development course. This site was built with custom HTML and CSS, some Bootstrap, jQuery, and JavaScript. Since turning this in and completing the course, I have continued to refine this project to enhance functionality, responsiveness, and accessibility.

  • I started work on this project back in late 2015/early 2016, but did not complete it until 2020. Most of that time it spent gathering dust. When I picked it up again, I revised and redesigned it. Initially, it was going to be a three page site, which I whittled down to one page to focus more on the music. Given the nature of the band and the aesthetic, I decided to lean into the mystery aspect and allowing the user to discover what the page had to offer, like the band biography and the music player in the radio. There is a JavaScript prompt that will appear if user interaction is not detected after a period of time to provide them with hints on what can be found on the page. This utilizes HTML, CSS, JavaScript, and jQuery.
